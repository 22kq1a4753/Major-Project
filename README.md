# Java Web Major Project

## Project Title
A Blockchain-Based Approach for Drug Traceability in Healthcare Supply Chain

## Technology Stack
- Java
- JSP & Servlets
- MySQL
- Apache Tomcat 9

## How to Run the Project
1. Clone the repository from GitHub
2. Open Eclipse IDE
3. Import the project as Dynamic Web Project / Maven Project
4. Configure Apache Tomcat 9
5. Import database from the SQL file
6. Update database credentials
7. Run the project on Tomcat server

## Note
This project is designed to run locally using Apache Tomcat and MySQL.
